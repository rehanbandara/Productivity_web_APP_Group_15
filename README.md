The Student Productivity Web App is a desktop-only academic productivity platform that integrates task management, study notes, and focus tools into one unified system. It helps university students plan daily routines, organize notes, and stay focused with a Pomodoro-style timer. Featuring a clean academic UI, intelligent task prioritization, and user-specific dashboards, the app acts as a personal academic planner designed to reduce stress, improve time management, and boost productivity.

The system integrates:
- Task management  
- Study note organization  
- Focus and productivity tools
